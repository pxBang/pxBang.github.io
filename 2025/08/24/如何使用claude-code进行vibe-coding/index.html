前言

本篇博客旨在分享如何使用Claude Code进行vibe coding的整体思路及关键步骤。

plan mode + CLAUDE.md

项目初期，不要急于开始编写代码。除非你对所需的代码细节了如指掌，否则应该首先使用Claude Code的plan mode生成一份CLAUDE.md文档。这份文档应包含以下内容：

  1. 项目代码结构（适用于非零起步的项目）
  2. 开发需求，需要详细到流程图级别（建议与大模型反复讨论打磨，以获得理想的流程图）
  3. 开发风格，例如面向对象编程等
