前言

背景：最近工作很忙，经常9、10点回家，晚一点甚至到11、12点。因此高强度工作下，如何保证睡眠成了关键问题

回家之后玩手机

回家之后可以玩手机，但是需要保证在11:30之前睡觉，给1个小时入眠时间，这样实际睡眠时间可以达到8个小时，才可以保证睡眠充足

回家之后玩电脑

回家之后可以玩电脑，但是需要保证在11:30之前熄灯睡觉，给1个小时入眠时间，这样实际睡眠时间可以达到8个小时，才可以保证睡眠充足

周六晚上

周六晚上可以稍微晚于11:30，但不应和平时有较大波动，否则会作息不规律

其余晚上

除了周六的其余六天，即使是周天晚上也应该在11:30熄灯睡觉

尽量带耳机

防止楼上咚咚声音影响入眠效率

使用家里的3060显卡-解决方案

发表于 2024-12-07 更新于 2025-08-24 本文字数： 365 阅读时长 ≈ 1 分钟

前言

本篇文档的背景是我希望连接家里的windows主机，并使用3060跑一些ai模型

目前提供两种解决方案

  1. 互联网远程连接
    1. 智能插座
    2. 向日葵远程控制
  2. 局域网远程连接
    1. 内网穿透
    2. 智能插座
    3. 微软远程桌面

共同点

  1. 都需要购买一个智能插座，小米智能插座3目前69元
  2. 都可以远程连接我的windows主机，在图形化界面进行操作

不同点

互联网远程连接

  • 优点
    1. 无需内网穿透，操作更简单
    2. 互联网连接，稳定连接，不会出现连接不上的情况
  • 缺点
    1. 走的是向日葵的服务器，因此网速带宽较低，可能会出现图形化界面模糊掉帧
    2. 不提供文件传输功能（代验证）

局域网远程连接

  • 优点
    1. 内网穿透后，直连互联网，网速较快（网速取决于windows的上传速度和客户端的下载速度）
    2. 内网穿透后，表面上类似于在同一局域网内，可以使用各种命令行进行通信，也可以使用ssh连接，scp各种协议都可以使用
  • 缺点
    1. 需要内网穿透，操作复杂一些
    2. 内网穿透不太稳定，nat套娃过多可能会穿透失败，需要有公网ip

确立周末放纵日和工作日的隔离关系

发表于 2024-09-09 更新于 2025-08-24 本文字数： 361 阅读时长 ≈ 1 分钟

前言

本篇文档的目的是确立周末放纵日和工作日的隔离关系，目的是让周末的放纵不影响工作日的作息

为什么周末放纵日会影响工作日的作息

周末白天的放纵不会影响工作日丝毫，但是周末晚上的晚睡会影响个人生物钟。

什么是周末放纵日对工作日的作息的影响

举个例子

  1. 周五晚上转钟2点睡觉，周六晚上转钟2点睡觉
  2. 周天晚上你想12点睡觉，周一上班有个好精神
  3. 做不到！！！，你周天晚上12点睡不着，因为前两晚都是转钟2点睡觉，你的身体在周天12点还没有进入睡眠状态
  4. 所以失眠，所以影响工作日上班的精神状态
  5. 不仅影响一晚，可能会持续几晚，因为生物钟的改变需要2、3天

怎么做来确立周末放纵日和工作日的隔离关系

  1. 周五晚上和周六晚上也12点睡觉，保证周末的作息和工作日的作息一致
    • 最终目的是保持恒定的生物钟
  2. 坚持周末白天自由放纵的原则不动摇
    • 该隔离关系不影响周末白天活动的丝毫
    • 周末白天自由放纵可以给人体“充能”

深入认识工资和五险一金——功能、使用方法、使用场景

发表于 2024-08-03 更新于 2025-08-24 本文字数： 1.9k 阅读时长 ≈ 6 分钟

前言

作为一个打工人，我辛辛苦苦付出脑力、体力、时间，换来的收益中，最重要最有用的就是工资和五险一金，那么提高对工资和五险一金的认识，用好工资和五险一金就十分重要。得到了工资和五险一金并不意味的收益的结束，距离你用好工资和五险一金还有很远的距离。

本篇博客的目的是为了深入认识工资和五险一金，从功能、使用方法、使用场景三个方面理解工资和五险一金。

工资

工资暂且不说。等发工资了之后补充该部分内容

住房公积金

功能

  1. 购房贷款：住房公积金账户中的余额可以作为购房的首付款，或者申请公积金贷款。相比商业贷款，公积金贷款的利率较低，可以减轻购房者的还款压力。

  2. 租房支出：如果没有购房计划，可以提取住房公积金用于支付房租，特别是对于在大城市工作的年轻人来说，这项功能非常实用。

  3. 房屋装修：一些地方的公积金政策允许提取住房公积金用于房屋装修，这样可以提升居住环境。

  4. 大修或重建住房：对于已经拥有住房但需要大修或重建的家庭，住房公积金也可以提供资金支持。

  5. 退休后提取：退休后，未使用的住房公积金可以一次性提取，作为养老金的补充。

使用方法

当然可以。以下是针对每个功能的住房公积金使用方法：

  1. 购房贷款：
    • 条件：需连续正常缴存住房公积金达到一定期限（通常为6个月至1年，各地规定不同）。
    • 步骤：
      1. 提交购房申请材料，包括购房合同、身份证明、收入证明等。
      2. 向住房公积金管理中心提出公积金贷款申请。
      3. 经过审批后，签订贷款合同。
      4. 贷款资金直接划入售房单位账户，用于支付购房款。
  2. 租房支出：
    • 条件：需无自有住房，并且连续正常缴存住房公积金达到一定期限（通常为3个月至6个月）。
    • 步骤：
      1. 准备租房合同、身份证明、收入证明等材料。
      2. 向住房公积金管理中心提交提取申请。
      3. 审核通过后，提取公积金用于支付房租，提取额度及频率由当地公积金政策规定。
  3. 房屋装修：
    • 条件：已购自住房屋，并且连续正常缴存住房公积金达到一定期限。
    • 步骤：
      1. 准备房屋产权证、装修合同、身份证明等材料。
      2. 向住房公积金管理中心提交提取申请。
      3. 审核通过后，提取公积金用于支付装修费用。
  4. 大修或重建住房：
    • 条件：房屋已达到需要大修或重建的标准，并且连续正常缴存住房公积金达到一定期限。
    • 步骤：
      1. 准备房屋产权证、大修或重建的相关证明材料、身份证明等。
      2. 向住房公积金管理中心提交提取申请。
      3. 审核通过后，提取公积金用于支付大修或重建费用。
  5. 退休后提取：
    • 条件：达到法定退休年龄，并且停止缴存住房公积金。
    • 步骤：
      1. 准备退休证明、身份证明等材料。
      2. 向住房公积金管理中心提交提取申请。
      3. 审核通过后，一次性提取账户内的全部余额。

使用场景

当然可以，以下是根据上述5点列出的对应使用场景：

  1. 购房贷款：
    • 场景：小李在武汉工作已有两年，打算在当地购买一套新房。他缴存住房公积金已满一年，于是向住房公积金管理中心申请了购房贷款，以较低的利率减轻了购房压力。
  2. 租房支出：
    • 场景：小张刚毕业后在北京找到工作，但暂时没有购房计划。他每月缴存的住房公积金已经达到了当地的提取标准，因此他向公积金管理中心申请提取公积金，用于支付每月的房租，缓解了租房的经济压力。
  3. 房屋装修：
    • 场景：小王购买了一套二手房，房屋状况一般，需进行装修。他向住房公积金管理中心提交了房屋产权证和装修合同等材料，提取了部分公积金，用于支付装修费用，改善了居住条件。
  4. 大修或重建住房：
    • 场景：小刘的老家房屋年久失修，需要进行大修。由于他已经连续缴存住房公积金多年，于是他向公积金管理中心申请提取公积金，支付房屋大修费用，确保房屋的安全和舒适。
  5. 退休后提取：
    • 场景：小赵今年正式退休，按照规定，他停止缴存住房公积金。他向公积金管理中心提交了退休证明和身份证明，顺利提取了账户内的全部公积金余额，作为退休后的经济补充。
